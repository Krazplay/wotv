WotV Data Tables

Github project page: https://github.com/krazplay/wotv
The tables are created using javascript, DataTables, and the datamine from shalzuth's Github.

You can temporarily move columns by drag and dropping their name (the first header), you can also move rows if you drag and drop the first cell of the row (there is a lag but it's working)
I've made many changes and has only applied a quick fix to old tables, they'll be reworked soon.
Added a link to the not yet written Crafting guide, but the simulator is there, still lack a few features but it's working.
